# Crypto Analyst AI

## 📌 Descrição
O **Crypto Analyst AI** é um sistema inteligente para análise de criptomoedas, reunindo dados de mercado e notícias em tempo real, com suporte de inteligência artificial para gerar análises claras e acessíveis.

## 🎯 Objetivo
Centralizar informações do mercado cripto e oferecer análises rápidas e organizadas para investidores iniciantes e intermediários.

## 🚀 Funcionalidades previstas
- Consulta de preços em tempo real
- Comparação entre diferentes ativos
- Resumo diário do mercado
- Histórico de consultas
- Lista de favoritos
- Futuramente: alertas personalizados, dashboards interativos e recomendações com IA

## 🛠️ Tecnologias
- Python
- FastAPI
- Banco de dados relacional (PostgreSQL)
- Integração com APIs externas (CoinGecko, CoinMarketCap, etc.)
- Modelos de IA para interpretação de notícias e geração de análises

## 📂 Estrutura inicial
- `docs/` → documentação
- `src/` → código fonte
- `tests/` → testes automatizados
